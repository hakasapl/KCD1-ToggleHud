# ToggleHud - KCSE

KCSE mod for Kingdom Come Deliverance 1 to toggle hud visibility with a hotkey at an engine level

## Building

1. Download and install Visual Studio 2026 with the Desktop Development for C++ Module
1. Open the Developer Powershell for VS18
1. Navigate to this directory and run the build script `.\buildRelease.ps1`. Output will be in the dist folder
